### ゲーム制作

#### 問題 19.

キャラクタがジャンプ出来る機能を付けて下さい。
一度ボタンを押すと特定の高さまで上昇し、再度落下する機能です。
ジャンプ中に再度ジャンプは出来ないようにして下さい。

#### 問題 20.

ブロックが地面についた時、マップの一部になるようにして下さい。
下記の様な、push メソッドを使用すると、
配列の最後にデータを追加出来ます。
これを応用して作成して下さい。

```ruby
array = []

array.push(1)
puts array #=> [1]

array.push(2)
puts array #=> [1, 2]
```
